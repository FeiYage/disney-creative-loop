# 迪士尼创意引擎 · 自动头脑风暴

基于华特·迪士尼创意策略的三位一体自动循环头脑风暴 Skill，适用于 QClaw 平台。

## 功能说明

用户只需提供一个点子/想法，该 Skill 会自动驱动三个 AI 角色协作：
- **梦想家**：天马行空描绘终极愿景
- **实干家**：制定具体可执行的行动计划
- **批评家**：从多角度审视方案并打分

三个角色自动循环迭代，直到方案评分达到 9.0 分以上（满分 10 分），全程无需用户中途参与。

## 触发关键词

脑暴、头脑风暴、创意方案、帮我策划、点子打磨、迪士尼

## 使用示例

在 QClaw 中输入：

> 脑暴：帮我想一个让公司 200 人积极参与读书打卡的点子。

Skill 会自动完成全部脑暴流程并输出最终方案报告。
